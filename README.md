# Free Fire Headshot Guide / የፍሪ ፋየር ሄድሾት መመሪያ

ይህ አፕሊኬሽን በፍሪ ፋየር (Free Fire) ጌም ውስጥ እንዴት ሄድሾት (Headshot) መምታት እንደሚቻል የሚያስተምር መመሪያ ነው። መመሪያው በእንግሊዝኛ እና በአማርኛ የተዘጋጀ ሲሆን፣ ለጀማሪዎችም ሆነ ለነባር ተጫዋቾች ጠቃሚ የሆኑ ቴክኒኮችን ያካተተ ነው።

## ገፅታዎች (Features)
- 🎮 **የሽፋን ገጽ (Cover Page):** ማራኪ የሆነ አቀባበል እና ወደ መመሪያው መግቢያ
- 📖 **ሁለት ቋንቋ (Bilingual):** ሁሉም ምዕራፎች በእንግሊዝኛ እና በአማርኛ ቀርበዋል
- ⚙️ **የሴንሲቲቪቲ ቅንብሮች:** ትክክለኛ የSensitivity ቁጥሮችን ያሳያል
- 🎯 **የድራግ ቴክኒክ (Drag Technique):** እንዴት ወደ ላይ በመሳብ ሄድሾት እንደሚመታ ያብራራል

## ቴክኖሎጂ (Tech Stack)
- Flutter (Dart)

## ማሄጃ (How to Run)
1. Flutter መጫኑን ያረጋግጡ
2. `flutter run` ብለው ትዕዛዝ ይስጡ
